## üì¶ Installation

<Steps>
  <Step title="Prerequisites">
    - Python 3.8+ 
    - `uvx` (recommended) or `pip`
    - Your NIA API key from [app.trynia.ai](http://app.trynia.ai/)
  </Step>
  
  <Step title="Install the Package">
    <CodeGroup>
      ```bash Option A: Using uvx (Recommended)
      # uvx will automatically handle the installation when you run it
      # No need to pre-install if you have uvx
      ```
      
      ```bash Option B: Using pip (If needed)
      pip install nia-mcp-server
      ```
    </CodeGroup>
  </Step>
  
  <Step title="Configure Your MCP Client">
    Choose your coding assistant below for specific configuration steps
  </Step>
  
  <Step title="Replace Your API Key">
    Get your API key from [app.trynia.ai](http://app.trynia.ai/) and replace `YOUR_API_KEY` in the configuration
  </Step>
  
  <Step title="Restart Your Client">
    Restart your coding assistant to load the new MCP server
  </Step>
</Steps>

## ‚öôÔ∏è Configuration for Your Coding Assistant

<Tabs>
  <Tab title="Cursor">
    <Steps>
      <Step title="Open MCP Configuration">
        Open your Cursor MCP configuration file:
        - **macOS**: `~/.cursor/mcp.json`
        - **Windows**: `%APPDATA%\Cursor\mcp.json`
        - **Linux**: `~/.config/cursor/mcp.json`
      </Step>
      
      <Step title="Add NIA Configuration">
        ```json
        {
          "mcpServers": {
            "nia": {
              "command": "uvx",
              "args": ["nia-mcp-server"],
              "env": {
                "NIA_API_KEY": "YOUR_API_KEY",
        ```

## ‚ö†Ô∏è Troubleshooting

<AccordionGroup>
  <Accordion title='"No API key provided"'>
    **Solution:** Make sure you've replaced `YOUR_API_KEY` with your actual API key from [app.trynia.ai](http://app.trynia.ai/)
    
    Double-check the environment variable is correctly set in your MCP configuration.
  </Accordion>
  
  <Accordion title='"Command not found: nia-mcp-server"'>
    **Solution:** Try installing with pip first:
    ```bash
    pip install nia-mcp-server
    ```
    
    Or ensure `uvx` is properly installed and in your PATH.
  </Accordion>
  
  <Accordion title='"Invalid API key"'>
    **Solution:** 
    - Double-check your API key at [app.trynia.ai](http://app.trynia.ai/)
    - Ensure there are no extra spaces or characters
    - Verify the API endpoint is set to `https://apigcp.trynia.ai/`
  </Accordion>
  
  <Accordion title='"Rate limit exceeded"'>
    **Solution:** You've used your 25 free requests. Check your usage and upgrade your plan at [app.trynia.ai](http://app.trynia.ai/)
  </Accordion>
  
  <Accordion title="MCP Server Not Responding">
    **Solution:**
    1. Check if the server is running properly
    2. Verify your MCP client configuration
    3. Restart your coding assistant
    4. Check the console for error messages
  </Accordion>
</AccordionGroup>
