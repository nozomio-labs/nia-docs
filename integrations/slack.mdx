<Note>
  This guide explains how to set up Nia AI in your Slack workspace and use it to interact with your codebase directly from Slack.



  * ***You have to index your repository on the*** [***webapp***](https://app.trynia.ai/) ***first.***
</Note>

## Installation

<Steps>
  <Step title="Add to Slack">
    1. Visit [Nia AI Dashboard](https://app.trynia.ai/settings/code-providers)

    2. Click "Add to Slack" button

    3. You'll be redirected to Slack's authorization page
  </Step>

  <Step title="Authorize Access">
    Select the channels where you want to use Nia AI:

    * Choose specific channels

    * Or allow access to all channels
  </Step>

  <Step title="Complete Setup">
    After installation:

    1. Nia AI bot will be added to your workspace

    2. You'll be redirected back to dashboard

    3. You can start using Nia AI in Slack
  </Step>
</Steps>

## Usage

<Tabs>
  <Tab title="Basic Commands">
    ```bash
    # Get help
    /nia

    # Set active project
    /nia list - list your available indexed projects

    # Choose project to chat with
    /nia use <project_id> - choose which project to chat with

    # Ask questions
    /nia chat <question> - ask a question about your current project
    ```
  </Tab>

  <Tab title="Mentions">
    ```bash
    # Mention the bot in any channel
    @Nia How do I implement the login flow?

    # Use in threads
    @Nia Can you explain this code?
    ```
  </Tab>
</Tabs>

## Channel Configuration

<Steps>
  <Step title="List Projects">
    ```bash
    /nia list
    ```

    Sets the active project for the current channel
  </Step>

  <Step title="Verify Setup">
    ```bash
    /nia use <project_id>
    ```

    Confirms the project is properly configured
  </Step>

  <Step title="Start Using">
    ```bash
    /nia chat <question>
    ```

    Begin asking questions about your code

## FAQ

<AccordionGroup>
  <Accordion title="Can I use Nia AI in private channels?">
    Yes, you can add Nia AI to private channels. Just invite the bot using `/invite @Nia`
  </Accordion>

  <Accordion title="How do I change projects?">
    Use `/nia use [project-id]` to switch to a different project in any channel
  </Accordion>

  <Accordion title="Are my code queries private?">
    Yes, all queries are processed securely and not stored permanently
  </Accordion>
