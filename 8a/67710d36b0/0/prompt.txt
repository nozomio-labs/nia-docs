Changelog for Feb 25, 2026

  1) slack search is now first-class

slack is fully integrated across connect → index → search. You can connect workspaces in Integrations, choose channels to index, search/read/grep Slack content, and manage reconnect/ cancel/delete flows more reliably.

  2) faster, safer search flows

improved search responsiveness with request cancellation, timeout fallbacks, and better payload validation. 

  3) home + onboarding overhaul

new Home experience and navigation defaults, better onboarding progression/resume behavior, improved org setup, and a cleaner agent-skill setup path with API key generation/copy UX improvements.

  4) activity + observability upgrades

activity feed now includes richer response snippets, source info, clearer query-type labels/icons, and Oracle job events.
