## Setup in 3 Steps

<Steps>
  <Step title="Get Your API Key">
    Visit [app.trynia.ai](https://app.trynia.ai) and sign up to get your free API key.
    
    <Info>
      You'll get 3 free index jobs to start - no credit card required!
    </Info>
  </Step>
  
  <Step title="Install Nia MCP Server">
    Open your terminal and install the Nia MCP server globally:
    
    ```bash
    pipx install nia-mcp-server
    ```
    
    <Note>
      Don't have pipx? Install it first with `pip install pipx` or uv!
    </Note>
  </Step>
  
  <Step title="Run Setup Command">
    Open your global terminal, run the setup command with your API key:
    
    ```bash
    pipx run --no-cache nia-mcp-server setup YOUR_API_KEY
    ```
    
    Replace `YOUR_API_KEY` with the key you got from step 1.
    
    <Tip>
      This command will automatically configure Nia MCP in cursor's mcp.json file.
    </Tip>
  </Step>
</Steps>
