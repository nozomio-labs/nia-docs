  <Accordion title="How are API calls counted?">

    * Asking questions about code

    * Indexing repositories

    * Using integrations

    * Running analysis
  </Accordion>

  <Accordion title="What happens if I exceed my API limit?">
    Free plan: Access paused until next month
    Pro plan: Charged $0.10 per additional call
